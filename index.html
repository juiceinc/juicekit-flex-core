JuiceKit Flex Core Library by juiceinc

To learn more about JuiceKit, visit the JuiceKit project page.

Getting Started

To use this library, you can compile the source code from scratch, download a compiled swc, or include a reference to juicekit-flex-core in your Maven pom.xml file.

To compile from scratch, go to building from source.

You can download a compiled swc here.

To use the following 

    
      com.juicekit.ui
      juicekit-flex-core
      2.0
      swc

Dependencies

This library depends on Flex 4.

Building from source

You can download this project in either zip or tar formats.

You can also clone the project with Git by running: 

$ git clone git://github.com/juiceinc/juicekit-flex-core

Follow the instructions in the README file to build this project as a library.

get the source code on GitHub : juiceinc/juicekit-flex-core
